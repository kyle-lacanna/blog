This week I progressed my Javascript abilities and learned how to make a slider for images. This type of display could be useful for any type of store, showcase, or even ads. I made a simple one with next and Previous buttons, but in the future I would like to expand it to automatically scoll through after a given time. I had a lot of fun with this and it was alittle challenging.


The slider works by assigning each image and index in a sort of array. Then when the next or prev button is pressed the count var is updates to only show the current image by translating the x direction attribute in the array of images.
